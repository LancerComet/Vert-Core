## 特征：

 - [x] 基于 TypeScript.
 
 - [x] 基于自有的一套开发规范与设计模式.
 
 - [x] 面向对象的设计方式.
 
 - [x] 关注运行时的数据安全，可创建运行时的类型安全实例.
 
 - [x] 锁定所有第三方依赖 Npm Package 版本.
 
 - [x] 含有 Store (Vuex), Router (Vue-Router).
 
 - [x] 提供诸如 Http、LocalStorage、Logger 等内部服务.
 
 - [ ] 两种使用方式：

   - 使用 Vert 的模板.

   - 将 Vert 作为类库使用.

 - [ ] 插件系统.
